# MyEdgeStyle

Edge 扩展：自定义新标签页（时间、天气、壁纸、毛玻璃镶边）。

## 功能

- **新标签页**：可选时间、天气（默认 Open-Meteo 免 Key，默认地区南京）、搜索（默认 Bing）、壁纸切换、顶部/底部毛玻璃镶边与透明度，设置保存在扩展选项页。

## 安装（开发者）

1. 打开 Edge，地址栏输入 `edge://extensions/`。
2. 开启「开发人员模式」。
3. 点击「加载解压缩的扩展」，选择本仓库根目录 `MyEdgeStyle`。
4. 新开标签页即可看到自定义页；点击扩展图标可打开设置。

## 图标

`assets/icons/` 下需有 16.png、48.png、128.png。当前为占位图，可替换为自有图标。

## 天气

默认使用 [Open-Meteo](https://open-meteo.com/) 免 Key API，默认地区为南京。可在扩展设置中切换天气地区及搜索引擎。

## Edge 默认新标签页

在扩展「设置」中勾选「使用 Edge 默认新标签页」并保存后，新开标签页会跳转到 `edge://newtab`。
